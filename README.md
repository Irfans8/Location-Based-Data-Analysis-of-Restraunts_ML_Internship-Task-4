Location-based Restaurant Analysis
📌 Project Overview
This project is part of my Machine Learning Internship at Cognifyz Technologies. The objective of Task 4 is to perform a comprehensive geographical analysis of the restaurant dataset to identify spatial trends and location-based performance metrics.

🎯 Objective
To analyze the distribution of restaurants across different cities and countries and investigate the correlation between a restaurant's location and its aggregate rating.

🛠️ Key Analysis Steps
City Distribution: Identified the cities with the highest concentration of restaurants.

Rating Correlation: Analyzed how average ratings vary from one city to another to determine if location influences quality.

Geographical Mapping: Plotted restaurant coordinates (Latitude & Longitude) to visualize the global reach and density of the dataset.

Statistical Insights: Used grouping and aggregation to identify the top-performing cities by rating.

📊 Visualizations Included
Bar Charts: Visualizing the top 10 cities with the most restaurants.

Performance Charts: Comparing the average aggregate ratings across different urban centers.

Geographical Scatter Plot: Mapping the physical location of every restaurant in the dataset using a coordinate-based heat map.

💻 Technologies Used
Language: Python

Libraries: * Pandas: For data aggregation and location grouping.

Matplotlib & Seaborn: For creating distribution plots and geographical scatter maps.

📂 Dataset Features Used
City

Longitude & Latitude

Aggregate rating

Restaurant Name

🚀 How to Run the Project
Ensure the Dataset .csv file is in your directory.

Install dependencies:

Bash
pip install pandas matplotlib seaborn
Run the analysis script:

Bash
python task4_location_analysis.py
📈 Key Insights
The analysis revealed which cities serve as major culinary hubs.

A clear distinction was found in average ratings based on geographical regions, suggesting that certain cities have a higher density of "Excellent" rated dining spots.
